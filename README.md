## Prerequisites

Install runtime at [https://bun.sh](https://bun.sh).

Install dependencies

`bun install`

## Develop

`bun run dev`

## Build

`bun run build`

## Deploy

This repository deploys automatically to Netlify at [https://dndbanner.netlify.app](https://dndbanner.netlify.app).
